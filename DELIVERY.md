## FREQUENTLY ASKED QUESTIONS

**Q: Do I need to change anything before running?**
A: No! The .env file already has your API key configured.

**Q: Can I use a different model?**
A: Yes! Edit .env and change GEMINI_MODEL to another model name.

**Q: What if the model isn't available?**
A: Run `python list_available_models.py` to see available models for your API key.

**Q: Can I use the JSON output in my own code?**
A: Yes! The `parse_json_from_response()` function returns a Python dict you can use.

**Q: Is the API key secure?**
A: The .env file is in your local project. Don't commit it to git if sharing publicly.

**Q: Can I improve the analysis prompts?**
A: Yes! Edit TEXT_REPORT_PROMPT and JSON_REPORT_PROMPT in app.py for different analysis.

**Q: What if Gemini misidentifies something?**
A: Visual-only analysis has limitations. Always verify with qualified engineers.
